# Cálculo IV

Repositorio académico con notas, material visual y recursos de apoyo para el curso de **Cálculo IV**.

El contenido está orientado al estudio de cálculo vectorial y geometría en varias variables, incluyendo teoría, interpretación geométrica, ejemplos y material didáctico complementario.

---

## Contenido actual

Actualmente el repositorio incluye notas y recursos sobre los siguientes temas:

- Sistemas de coordenadas
- Parametrización de superficies
- Teorema de Stokes
- Teorema de Gauss

---

## Material didáctico

Además de las notas teóricas, el repositorio contiene:

- Infografías del Teorema de Gauss
- Infografías del Teorema de Stokes
- Árbol de decisiones para el uso de los teoremas de:
  - Green
  - Stokes
  - Gauss
- Diferencia entre frontera de una superfice y una región en $\mathbb{R}^3$
- Uso y aplicaciones sobre el teorema de Gauss.

---

## Objetivo del repositorio

Este proyecto busca reunir material académico claro, visual y accesible para estudiantes de licenciatura en matemáticas, física e ingeniería interesados en el cálculo vectorial y la geometría.

El repositorio continuará creciendo con:

- Nuevas notas
- Ejercicios resueltos
- Aplicaciones físicas
- Material visual
- Referencias bibliográficas
- Recursos interactivos
