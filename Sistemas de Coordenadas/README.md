# Sistemas de Coordenadas

En estas notas hablaremos sobre algunos de los sistemas de coordenadas más utilizados en cursos de cálculo vectorial y geometría en varias variables.

El objetivo de este material es presentar de manera geométrica e intuitiva las coordenadas polares, cilíndricas y esféricas, así como sus relaciones con el sistema cartesiano y algunos de los errores conceptuales más comunes al trabajar con estos sistemas.

---

## Contenido

Este documento incluye:

- Coordenadas polares
  - Interpretación geométrica
  - Conversión entre coordenadas polares y cartesianas
  - Ejercicios básicos

- Coordenadas cilíndricas
  - Construcción geométrica
  - Relación con coordenadas polares
  - Cambio a coordenadas rectangulares

- Coordenadas esféricas
  - Interpretación geométrica de los parámetros
  - Conversión a coordenadas cartesianas
  - Aplicaciones geométricas y físicas

- Errores comunes
  - Confusión entre parámetros
  - Interpretación incorrecta de superficies coordenadas
  - Errores frecuentes en cambios de coordenadas

---

## Objetivo

Estas notas están pensadas como apoyo para cursos de:

- Cálculo Vectorial
- Cálculo en Varias Variables
- Geometría
- Física Matemática

El enfoque busca priorizar:

- la interpretación geométrica,
- la visualización espacial,
- y la comprensión conceptual de los cambios de coordenadas.
